# python打印图形

在一些python面试的时候，有时候会让你打印一些图形，比如菱形之类的。本文就搜集一些这方面的面试题，同时把答案也放在这里。

<!--more-->

**【1】一维整形数组[2, 5, 6], 请每行打印2个\*， 5个\*， 6个\***
这个题目非常简单，python支持`字符 * 数字`的方式重复打印字符，所以这个题目就非常简答了。
代码如下： 
```python
_list = [2, 5, 6]
for i in _list:
    print "*" * i
```

**【2】一维整形数组[2, 5, 6], 请每列打印2个\*， 5个\*， 6个\***
跟上一题类似，但是变成每列打印，这就有点难度了。先看代码：
```python
_list = [2, 5, 6]

height = max(_list)

for i in range(height, 0, -1):
    line = []
    for j in _list:
        if j >= i:
            line.append("*")
        else:
            line.append(" ")
    print "".join(line)
```
解这道题的思路要想像我们在盖房子，这个房子有几个单元，就是列表的长度。但是这个楼有一个特点，每个单元楼的高度是不一样的。最高的单元楼就是列表中最大的那个元素，然后我们就从最高那层楼逐层开始盖房子（向下遍历），跟我们现实中盖房子不太一样。如果我这个单元楼的高度比当前楼层高，我就盖一层（打\*号），否则我就不盖（打空格），一直盖到一层为止。
**【3】设计一个打印菱形的函数**
这道题就能看到python语言的便利了，如果是其他语言的话，代码可能要多出一倍不止，这道题的关键就是算好数了，在纸上多画几遍就可以了。代码如下：
```python
def printPyramid(level):
    for i in range(level):
        print " " * (level - i - 1) + "*" * ((i + 1) * 2 - 1)

    for i in range(level - 1, 0, -1):
        print " " * (level - i) + "*" * (2 * i -1)

printPyramid(5)
```
